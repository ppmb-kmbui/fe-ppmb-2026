# PPMB KMBUI 2026 Frontend

Frontend website PPMB Keluarga Mahasiswa Buddhis Universitas Indonesia 2026.

## Development

```bash
npm install
npm run dev
```

Development server tersedia di `http://localhost:3000`.

## Quality check

```bash
npm run check
```

## Branch
- `main`: production dan baseline stabil.
- `development`: integrasi fitur.
- `feature/*`: pengerjaan fitur melalui pull request ke `development`.
